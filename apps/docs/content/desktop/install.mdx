omniUI ships with a companion Electron app. It runs locally, holds your keys, and lets the agent edit any project that has an `omniui.config.json` at its root.

## Quick start

```bash
git clone https://github.com/vineethpawar/omniUI
cd omniUI/apps/desktop
npm install
npm run start
```

The window opens on the Project tab. Pick the folder for the repo you want to polish. The folder must contain an `omniui.config.json` (see [Project config](/desktop/config/)).

After picking, drop your Anthropic key on the Settings tab. Then switch to Chat and send a task.

## What the agent can do

- Read any file inside your project root (or only inside `readPaths` if you set it).
- Write to files only inside `writePaths`. Anything outside is denied by the PreToolUse hook before the SDK touches it.
- Read pre-downloaded Figma YAML/JSON dumps offline; live-fetch any Figma URL via the REST API when you've set a Figma access token.

## Every Write or Edit is held for approval

Before any file is written, the agent emits a `write_pending` event. The renderer draws a card:

- File path at the top.
- For `Edit`: the matched `Was:` block and the proposed `Becomes:` block side by side.
- For `Write`: the full file content with a character count.
- Apply / Reject buttons.

Apply commits the write. Reject denies it and the agent continues without that change. If you ignore the card, the agent waits.

## Why a subprocess

The agent runs in a node subprocess spawned per chat turn. Two reasons for that:

1. The agent SDK is happiest with a Node runtime; isolating it from the Electron main process means a SIGTERM kills only the agent, not the window.
2. When the time comes to swap the agent runtime for a local model server, you only change one entry path.

## Common issues

| Symptom | Likely cause |
|---|---|
| "ANTHROPIC_API_KEY not set" | Drop a key on the Settings tab. Stored at `~/Library/Application Support/omniUI/settings.json` on macOS. |
| "No omniui.config.json found" | The chosen folder doesn't have a config at its root. See [Project config](/desktop/config/). |
| `"<path> is outside writePaths"` | Add the path to `writePaths` in your config and reload the project. |
| Window stays blank | First `npm install` may take a minute. Check the dev tools console for errors. |
